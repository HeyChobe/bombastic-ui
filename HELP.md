# Button

- **Use**

```javascript
import { Button } from "bombastic-ui";

<Button bg="primary" label="Primary" onClick={() => console.log("hola")} />;
```

- **Properties**
  - **_bg_** = primary | secondary | ok | cancel | warning | hex-color (#000000) : _string_
  - **_label_** : _string_
  - **_onClick_** : _() => void_
  - **_ghost_** : _boolean_
